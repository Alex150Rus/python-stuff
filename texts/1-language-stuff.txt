Ес­ли у тебя на ком­пе есть сов­ремен­ный дис­три­бутив Linux, то в него уже вхо­дит Python 3, 
а писать пер­вые прог­раммы будет удоб­но в IDLE — прос­том редак­торе кода, который при­лага­ется к Python. 
В Ubuntu, что­бы уста­новить его, набери в кон­соли
sudo apt-get install idle3

За­пус­тив IDLE, зай­ди в Options → Configure IDLE, перей­ди на вклад­ку General и пос­тавь галоч­ку 
на пун­кте Open Edit Window, наж­ми ОK и переза­пус­ти IDLE. Теперь мож­но писать прог­раммы, 
сох­ранять их и запус­кать кла­вишей F5.

--СПИСКИ--
a = [67, 5, 90, 20, 30]
a[2] - по индексу получаем значение
a.append(33) - добавляем элемент в конец списка
a[-1] - последний элемент списка
a.sort()

Срез - получение какой-то части списка, которая, в свою очередь, тоже является списком.
список[x:y:z]
x - номер элемента, с которого берётся Срез
y - последний элемент среза
z - интервал, через который мы берём значения (не обязательный)
a[1:4] - срез элементов списка с 1 до 3 (4-ый не включается в срез)
a[2:] - со второго и до конца
a[::2] - каждый второй элемент списка
a[::-1] - переворачивает список

Элементами списка могут быть другие списки. Чтобы обратиться к элементам списка внутри списка, 
используй ещё одни []
a = [[1, 2, 3], [44, 45, 46]]
Это двумерный список. Он напоминает таблицу. a[1][0] выведет 44 - первое число во второй строке
Список - изменяемая последовательность. Это значит, что если ты изменишь список, то тебе
не придётся его переопределять и заново сохранять в переменную.

Строка - неизменяемая последовательность.

Кортеж - ещё один неизменяемый тип данных. Это тот же список, но ты не можешь его изменять.
Так он занимает меньше памяти. a = (1, 2, 3, 4)

Множество - ещё одна последовательность элементов, каждый из которых не имеет своего индекса
Т.е. элементы не упорядочен и ты не можешь обратиться к ним по индексу. Зато все элементы
множества уникальны и не повторяются
Преобразуем имеющийся список в множество и увидем, что в нём не останется повторяющихся элементов.
l = [1, 2, 2, 3, 3, 4, 1]
m = set(l)
print(m) выведет set([1, 2, 3, 4]) - исчезли повторяющиеся элементы
Для превращения множества или чего другого в список list()
